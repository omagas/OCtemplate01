1. Installation Top

Extract the zip file using Winzip or Winrar on windows PC using the password provided on “Password” column under “My Downloadable Products” In “My Account”.

Copy the directories from “Opencart Files” (provided in to ZIP file) to your opencart root. Copy those directories at your opencart root respectively,
  e.g Your copy path will be
  “upload/admin”(Overwrite this folder)
  “upload/catalog”(Overwrite this folder)
  “upload/image”(Overwrite this folder)

If you want to enable languages then you can download it from this site http://www.opencart.com .
  Download language from given url and paste it to your language directory.
  Language directory root:"upload/catalog/language".

Now we have to set our Template from Admin panel
  Under Admin Panel ->System->Settings->Store->Template
  Select “OPC050112” into Templates field.
  And Click on “Save” button at top right corner.

2.Banner Settings Top

2.1.Main Banner

Add banner for slideshow

Now for adding new banners go to System->Design->Banners.
(disable HP Products, Manufacturers, Samsung Tab by clicking edit button)
Click on “Insert” button to add new banner.
Give banner name ,make status enable and click on “Add Banner” button.
Give title of your banner and click on image to upload your banner.
Again click on “Add Banner” button and upload your rest two banners.
Now click on “Save” button.

Uninstall Slideshow module  Go to  Extensions->Modules->Slideshow-> Uninstall.

Now, Install Templatemela Slideshow module  Go to  Extensions->Modules-> Templatemela Slideshow-> install(After install click on "Edit" link).

Now to activate your banner go to Extensions->Modules-> Templatemela Slideshow click on Add Module button.
From banner select your banner,
Give Width & Height (1000 X 393) (your banner width and height),
From the Layout select Home Page,
From Position select Content Top,
From status select Enable ,
Give short order=1.
Click on “Save” button.
Same way insert another slideshow.

11. Language Settings Top

We have provided Laguages with this package.
To Enable Laguages login to your admin panel and do follwing settings.

Go to  System->Localisation->Languages,
Click on “Insert” button to add new Language.
For Example To add dutch Language :
Language Name: Chinese,
Code: cn,
Locale: zh,zh-hk,zh-cn,zh-cn.UTF-8,cn-gb,chinese,
Image: cn.png,
Directory: chinese,
Filename: chinese,
Status: Enable,
Sort Order: as you like.
Click on “Save” button.

Same way you can do the settings for other Languages. Click here to view settings for other language.
